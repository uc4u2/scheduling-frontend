# Analytics How-To

## Where to click

Manager Portal → Services & Bookings → Analytics

## Step 1: Choose a report

1. Select a date range.
2. Pick a report or dashboard view.
3. Review totals and trends.

## Step 2: Filter results

1. Filter by staff, location, or service (if available).
2. Save or export results if needed.
