# Resources How-To

## Where to click

Manager Portal → Services & Bookings → Resources

## Step 1: Create a resource

1. Click **Add Resource**.
2. Fill the fields:
   - Name
   - Description
   - Capacity
3. Click **Save**.

## Step 2: Assign to services

1. Open a service in Services.
2. Add the resource under required resources.
3. Save changes.
