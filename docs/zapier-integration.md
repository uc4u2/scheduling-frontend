# Zapier Integration

Schedulaa connects to Zapier for automation and analytics workflows.

## Where to click

Manager Portal → Zapier

## Setup steps

1. Open **Zapier API keys** and create/copy a key.
2. Open **Event hooks** and paste your Zapier Catch Hook URL.
3. Choose the events you want Schedulaa to send (for example: booking.created, payroll.payment_requested).

## Common questions

- “Where do I connect Zapier?” → Manager Portal → Zapier.
- “Where do I add my Zapier webhook?” → Zapier → Event hooks.
