# Job Postings

Use Job Postings to publish open roles and track applicants.

## Where to click

Manager Portal → Overview → Job Postings  
Direct URL: `/manager/job-openings`

## What you can do

- Create and publish new job openings.
- Edit or pause existing postings.
- Review candidates tied to each role.

If someone asks “Where do I post a job?”, send them to this page.
