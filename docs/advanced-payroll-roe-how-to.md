# ROE How-To

## Where to click

Manager Portal → Advanced Payroll → ROE

## Step 1: Select the employee

1. Choose **Department** (optional).
2. Toggle **Show archived employees** if needed.
3. Choose the **Employee**.

## Step 2: Review employee info

1. Confirm the employee summary (hours, pay, last day, and location).
2. If anything looks wrong, fix the data in payroll before continuing.

## Step 3: Create a new ROE

1. Fill **Country**, **Province/State**, and **Employment Type**.
2. Enter **Last Day**, **Total Hours**, **Total Pay**, and **Reason**.
3. Click **Create ROE**.

## Step 4: Approve and export

1. In **Existing ROEs**, click **Approve**.
2. Use the **PDF** icon to preview the ROE.
3. Use the **XML** icon to export the ROE XML file.

## Step 5: Audit history

1. Click the **Audit Trail** icon to see changes and approvals.

Notes:
- ROE is a Business-plan compliance document.
- PDF/XML export is available after approval.
