# Manager Overview Shortcuts

The Overview section is a shortcut hub in the Manager Portal sidebar.

## Where to click

Manager Portal → Overview → (choose a shortcut)

## Shortcuts and what they do

| Shortcut | What it does |
| --- | --- |
| Team Activity | Snapshot of team status, coverage, and activity signals. |
| Master Calendar | Full calendar view across teams and resources. |
| Candidate Funnel | Recruiting pipeline and stage movement. |
| Job Postings | Create, edit, and publish job postings. |
| Recruiter Performance | Performance and throughput insights for recruiters. |
| Candidate Search | Search candidate records and open profiles. |
| Feedback Notes | Central log for manager notes and feedback. |
| Recruiter Availability | Availability view for recruiters and interview scheduling. |
| Recent Bookings | Recent bookings list and activity feed. |
| Payroll Audit History | Log of payroll actions and audit events. |
| Monthly Attendance Calendar | Attendance heatmap for the current month. |
| Candidate Profile | Detailed candidate profile view and history. |
