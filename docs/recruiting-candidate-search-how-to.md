# Candidate Search How-To

## Where to click

Manager Portal → Overview → Candidate Search

## Step 1: Set filters

1. Enter **Job Applied For**, **Recruiter ID**, or **Status**.
2. Click **Search**.

## Step 2: Review results

1. Review the table of candidates and conversion status.
2. Adjust filters and search again if needed.

Notes:
- This view is read-only and lists matching candidates.
