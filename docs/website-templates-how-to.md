# Website Templates How-To

## Where to click

Manager Portal → Website & Pages → Website Templates

## Step 1: Choose a template

1. Browse the template cards.
2. Click **Select** to focus a template.
3. Use the **Version** dropdown if multiple versions are available.

## Step 2: Preview

1. Click **Preview** to open the live preview dialog.
2. Switch **Device** (mobile, tablet, desktop).
3. Click page buttons to preview different pages.

## Step 3: Import

1. Click **Use** (or **Use this template** in the preview dialog).
2. Wait for the import confirmation.
3. Click **Start editing** to jump into the builder.

Notes:
- Import replaces pages with the selected template content.
- You can re-import another template at any time.
