# Recruiting How-To

This guide covers the core recruiting workflow in Schedulaa.

## Step 1: Create a job posting

1. Manager Portal → Overview → Job Postings.
2. Click **New job posting**.
3. Fill in title, location, and description.
4. Publish the posting.

## Step 2: Review candidates

1. Manager Portal → Overview → Candidate Funnel.
2. Open a stage to see candidates in that pipeline step.
3. Move candidates between stages as they progress.

## Step 3: Search candidate profiles

1. Manager Portal → Overview → Candidate Search.
2. Use filters or search by name/email.
3. Open a candidate to view the full profile.

## Step 4: Send invitations and forms

1. Employee Portal → Tabs → Invitations.
2. Send a candidate invite or open an existing one.
3. Choose the candidate experience:
   - attach a candidate form
   - send without form
   - set candidate document upload to hidden, optional, or required
   - attach questionnaires when needed
4. Use Candidate Forms for structured intake.

### Invitation rules

- `Save Defaults` stores recruiter-side invitation preferences. It is not the same as editing a form template.
- `Save Email Template` stores the email message for the profession.
- Candidate form templates control the candidate-facing intake structure.
- Core intake fields like name, email, phone, purpose, and reference link stay structured but can be:
  - hidden
  - optional
  - required
- Custom form fields are for extra questions beyond those booking basics.

### Candidate preview

The invitation page preview should be used as an operational check before sending.

It shows:
- visible core intake fields
- additional form fields
- document upload mode
- attached questionnaires
- hidden core intake fields

## Step 5: Schedule meetings

1. Employee Portal → Tabs → Upcoming Meetings.
2. Share links or confirm interview times.

## Common questions

- "Where do I post a job?" → Manager Portal → Overview → Job Postings.
- "Where do I move candidates through stages?" → Candidate Funnel.
