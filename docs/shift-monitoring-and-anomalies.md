# Shift Monitoring and Anomalies

Use these tools to watch live shifts, review time tracking, and investigate anomalies.

## Where to click

Manager Portal → Shifts & Availability

| Feature | Where to click |
| --- | --- |
| Shift Monitoring | Manager Portal → Shifts & Availability → Shift Monitoring |
| Time Tracking | Manager Portal → Shifts & Availability → Time Tracking |
| Fraud / Anomalies | Manager Portal → Shifts & Availability → Fraud / Anomalies |
| Swap Approvals | Manager Portal → Shifts & Availability → Swap Approvals |

## What each view is for

- Shift Monitoring: live view of shift activity and coverage.
- Time Tracking: review clock-ins/outs, approve time entries.
- Fraud / Anomalies: highlights unusual time patterns that may need review.
