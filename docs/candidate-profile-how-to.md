# Candidate Profile How-To

Use the Candidate Profile to review history, notes, and stage status for a candidate.

## Where to click

Manager Portal → Overview → Candidate Profile

## Step 1: Open a candidate profile

1. Go to Manager Portal → Overview → Candidate Search.
2. Search by name or email.
3. Click a candidate row to open the profile.

## Step 2: Review key sections

Common sections include:

- Contact details (name, email, phone)
- Current stage and pipeline history
- Notes and feedback
- Interview history / meetings (if scheduled)
- Attached forms or questionnaires

## Step 3: Update status

1. Change the candidate stage if they move in the pipeline.
2. Add manager notes or feedback.
3. Save changes.

## Step 4: Follow-ups

- Use Upcoming Meetings to confirm interview status.
- Check Candidate Forms and Questionnaires for intake responses.

## Related locations

- Candidate Funnel: Manager Portal → Overview → Candidate Funnel
- Candidate Search: Manager Portal → Overview → Candidate Search
- Invitations and Forms: Employee Portal → Invitations / Candidate Forms
