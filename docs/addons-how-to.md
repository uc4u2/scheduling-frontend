# Add-ons How-To

## Where to click

Manager Portal → Services & Bookings → Add-ons

## Step 1: Create an add-on

1. Click **Add Add-on**.
2. Fill the fields:
   - Name
   - Description
   - Price
   - Duration
3. Choose the **Parent Service** the add-on belongs to.
4. Click **Save**.

## Step 2: Edit or disable

1. Open the add-on row.
2. Update price or duration.
3. Toggle **Active** to disable.
4. Save changes.

## Step 3: Test on booking flow

1. Visit the public booking page.
2. Select the parent service.
3. Confirm the add-on appears as an upsell.
